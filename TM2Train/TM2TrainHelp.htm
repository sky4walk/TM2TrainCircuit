Rail Types

To differ between the different raile types there is a color value depending on the rail type on top left position in every image. All image transformation also change this position to another edge. There is also another color value which indicates the state of the switch. It indiceates which direction a train is leaving the switch if it comes from the above direction where the rail type marker is.

Connector:

   Start: Startposition shows direction

   Crossing: two rails crossing

  Straight: a connection between other rails

  Curve: Change of direction

Switches:

  FlipFlop set to True: a switch after every passing the direction changes

  FlipFlop set to false: a switch after every passing the direction changes

  Lazy set to true: when a train comes from left or down it changes the leaving direction

  Lazy set to true: when a train comes from left or down it changes the leaving direction

  Sprung set to true: it never will change the direction

  Sprung set to true: it never will change the direction
